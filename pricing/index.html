FAQ
Is mist free?

No, mist is a paid bot. If you want to test the bot out before you buy it, you can head over to our discord server to request a 1-day trial.

How can I pay for mist?

We accept the following payment methods: PayPal, Cashapp, Crypto (Bitcoin, Litecoin and Ethereum), Venmo, Debit/Credit Cards (Stripe) and Apple Pay. To purchase the bot, head over to our discord server.

Can I get a refund if I don’t want to use mist anymore?

You can check if you are eligible for a refund in our Discord server here.

Can I transfer one server whitelist to another?

We offer one server transfer for every whitelist. To get a transfer, open a ticket in our Discord server and provide a valid proof of purchase. Please note that we reserve the right to withhold any transfers if we deem necessary.

My previous account got disabled, can I still use mist?

Certainly, open a ticket in our Discord server and provide a valid proof of purchase. Please note that we reserve the right to withhold any transfers if we deem necessary.
